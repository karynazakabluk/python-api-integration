# Python & API

## Del 1
Programmet henter data fra et API om mennesker i verdensrommet

Det viser:
- hvor mange personer er i verdensrommet
- navn til dem
- på hvilket fartøy er de

## Del 2
Programmet henter posisjon til ISS fra API

Så:
- bruker latitude og longitude
- lager et kart med folium
- viser ISS som en markør på kartet

## Hvordan kjøre programmet

Installer bibliotek:
python3 -m pip install requests folium

Kjører programmene:
python3 program.py
python3 map.py
